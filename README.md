# Content Studio

**Transform ideas into impact.**

Content Studio is your professional workspace for crafting compelling content and producing polished documents — all powered by specialized AI skills.

## The Problem This Solves

You know the frustration. Great ideas get stuck between **strategy** and **execution**.

You craft brilliant copy in one tool, then manually format it into documents in another. Your SEO audit lives in a spreadsheet, your content strategy in a doc, your presentation slides in yet another app. Each handoff costs time. Each format change risks errors.

**Content Studio ends the fragmentation.**

Combine world-class marketing expertise with enterprise-grade document generation in one seamless workflow. Write landing page copy, then instantly transform it into a pitch deck. Audit your SEO, then generate a professional report. Plan your pricing strategy, then output a polished proposal.

## How It Works

### Two Phases, One Workflow

**Phase 1: Craft** — Use marketing skills to create strategy and copy

**Phase 2: Deliver** — Use document skills to format professional outputs

**Result:** Content that works brilliantly and looks professional too.

### See It In Action

```bash
# Step 1: Write landing page copy that converts
/copywriting
"I need compelling landing page copy for a new SaaS analytics tool"

# Step 2: Transform that copy into an investor pitch deck
/pptx
"Create a 10-slide pitch deck from the landing page copy"

# Step 3: Done. Professional documents saved to your directory.
```

### It's This Simple

```
/<skill-name> [your instructions]
```

**Try these now:**

- `/seo-audit https://yoursite.com`
- `/email-sequence Create a 5-email welcome sequence for new users`
- `/docx Write a professional proposal for a web design project`

## Get Started

### What You Need

Content Studio works with tools you may already have:

- **[Claude Code](https://claude.ai/code)** — Anthropic's AI CLI
- **OpenCode** — Compatible AI assistant

### Quick Setup

**Node.js packages** (install in project scope):

```bash
npm install <package>
```

**Python scripts** (using `uv` for speed):

```bash
uv run <script.py>
uv add <package>
```

**That's it.** No complex setup. No steep learning curve.
